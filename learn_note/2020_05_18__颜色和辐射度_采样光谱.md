# 采样光谱

采样光谱沿用了 参数光谱的基础架构，它是用一个均匀的空间，用 60 个样本，对 400-700 nm 进行采样

400-700 是人类最敏感的范围，我们进行 60 个样本，也是为了足够细

第一个空间是 [400, 405）第二个是 [405, 410)
