# 物体空间，世界空间，相机空间
- 物体空间： 以几何体中心为原点的坐标系
- 世界空间： 所有其他空间的标准空间
- 相机空间： 以相机点为原点的坐标系
- - Z轴：view direction 朝向向量
  - Y轴：上向量
  - X轴：右向量
  - 使用相机空间，来判断物体是否应该被看见
